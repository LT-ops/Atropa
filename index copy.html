About

The Atropa Token Family Explorer is a specialized tool for tracking tokens created by the token factory at address 0x394c3D5990cEfC7Be36B82FDB07a7251ACe61cc7 on the PulseChain network.

The token factory maintains parent-child relationships through the GetStandardTokenParent function, which takes a token address and returns its parent address.

This tool allows you to search for tokens created by wallet 0xBF182955401aF3f2f7e244cb31184E93E74a2501 and explore their parent-child relationships.
